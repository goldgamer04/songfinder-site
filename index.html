SongFinder

A Windows desktop tool for cataloguing tempo, key, and time‑signature of your favourite songs.

What is SongFinder?

SongFinder is a personal, non‑commercial utility that helps musicians, educators and music enthusiasts build a searchable database of songs annotated with BPM, time‑signature, key, and more.

  • Look up tracks online with a single click.
  • Store everything locally in a tiny SQLite file—no cloud account required.
  • Export to Excel for lesson plans, DJ sets, or set‑list prep.

How does it work?

The app uses the Spotify Web API’s Audio Features endpoint to retrieve tempo, key and meter metadata for commercially released recordings. It caches results offline so you’re not repeatedly calling the API.

Note: SongFinder never asks for your Spotify password. The first time you use the tool it opens the official Spotify login page so you can grant a temporary access‑token. You can revoke that permission at any time from your Spotify account settings.

Privacy Policy

SongFinder is a desktop‑only application. All song data—including any Spotify IDs—are stored locally on your own computer in a SQLite database file named songs.db. The app sends no personal or analytical information anywhere.

  • No account creation.
  • No third‑party trackers or ads.
  • You may delete the database at any time.
